# DIVINA GULA

## Description

This project is aimed at users who wants to make an order to Divina Gula in an easier and faster way instead of phone calls or order locally

### Prerequisites

- [List of prerequisites, e.g., PHP - 8.2.13, Node.js, Composer, Laravel 10.]

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/aencardenas/divina.git

2. Environment variables
    ```sh
    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=divina
    DB_USERNAME=root
    DB_PASSWORD=
3. Run migrations
    ```sh
    php artisan db:seed
4. Run development server
    ```sh
    npm run dev
5. Run server
    ```sh
    php artisan serve
